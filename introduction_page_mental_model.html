How this works

The core of our technology leverages MPC to enable third parties to compute authorized inferences without ever accessing the underlying raw financial data. Through our proprietary implementation of secure multiparty computation protocols, we create cryptographic circuits that precisely match your consent specifications.

Each inference request triggers a distributed computation protocol where the raw data remains encrypted and fragmented. Our TSS implementation ensures that signing keys for data access are never reconstructed in a single location, while our MPC protocol enforces that only pre-approved computational circuits can be executed.
