Privacy Consent Options Documentation

Explore different implementations of privacy consent interfaces. Each version presents a unique approach to user interaction and information presentation.

Description File Explanation Reasoning
Baseline with Superficial Choices View File Minimalist single-level display showing only main categories initially. Clicking reveals a full-page or modal view with function details. All functions for that category presented as descriptive text with no direct checkboxes for functions. Tests user response to a simplified consent interface that reduces cognitive load by limiting initial choices while still providing detailed information on demand.
Grid Card View View File Displays all categories as large cards in a grid layout. Each card shows category name and brief description, expanding to full width when clicked to reveal functions as smaller sub-cards. Evaluates how spatial organization and visual hierarchy through card sizing affects user comprehension and decision-making in privacy choices.
Progressive Accordion Hierarchy View File Categories expand downward when clicked, revealing indented function checkboxes below. Previous categories remain expanded as users open others, creating a growing, scrollable document. Tests user preference for a continuous, expanding document format that maintains context while progressively revealing more detailed options.
Stepped Wizard View File Shows one category at a time with all its functions, using Next/Previous navigation. Includes progress indicator and forces sequential decision-making with fully expanded functions for the current category. Assesses the effectiveness of a guided, linear progression through privacy choices, potentially reducing decision fatigue through structured progression.
Two-Column Matrix Hierarchy View File Features a two-column layout with categories in the left column and permanently visible but grayed-out functions in the right column. Selecting a category activates its related functions with clear visual grouping through spacing. Evaluates how maintaining full context visibility while using visual hierarchy affects user understanding and decision-making efficiency.
